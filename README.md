# City - Smart
plataforma de una ciudad inteligente desarollada por BeeStudents utilizando Tecnologias para Frontend, UI/UX y acceso a algunas API's publicas.

## Caracteristicas

- Login y Registro de usuarios.
- Dahboard responsivo.
- Integracion con Firebase (Pasos evoluticos).
- Consumo de API's Publicas(clima,Noticias,Calidad del aire).

## Tecnologia utilizada (Stack de desarollo)

- Html5.
- CSS3.
- Boostrap 5.
- Javascript.
- Firebase.
